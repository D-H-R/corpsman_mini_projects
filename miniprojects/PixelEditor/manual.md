# Usermanual for PixelEditor

This is the usermanual for Pixeleditor, mainly to show the "hidden" features like key combinations and other stuff.


## Mouse actions
| Action | Description |
| --- | ---
| wheel (up/down)| zoom (in/out)


## Keyboard actions
| Action | Description |
| --- | ---
| CTRL + N | New image
| CTRL + O | Options
| CTRL + S | Save
